# weApp-node_koa
一个小程序与公众号的项目
```
    我的目录结构是
    ----servers ---->后端代码
    ----config  ---->配置信息
    ----client  ---->小程序代码
    如果有同学clone代码，可能启动不了。
    1.config下面的https文件夹下面的证书我删除了，如果不需要只需要在www文件中注释https服务，
        如果有需要，只需要把证书放在https文件夹下，修改www里面加载证书的名称
    2.一些微信公众号和小程序的配置项我删除了，你们用你们的配置就行
    3.数据库配置我页删除了！你们根据需要使用，哈哈。
    如果有沟通可以加微信好友，聊聊
```
